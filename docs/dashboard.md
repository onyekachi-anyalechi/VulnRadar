# Dashboard

## View modes

- Show Watchlist Only (default)
- Show All Active Threats (CISA KEVs)

## Risk ranking

Sorted by: KEV > EPSS > CVSS.

PatchThis is treated as a critical signal and is prioritized above KEV/EPSS/CVSS.

The table includes `in_patchthis` and `priority_label`.
